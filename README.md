## 🎨 What Was Built

A fully browser-based air drawing app powered by **MediaPipe Hands AI** — no plugins, no install, just your webcam!

## ✋ Hand Gestures

| Gesture | What it does |
|---------|--------------|
| ☝️ Index finger up | Draw / Paint |
| ✌️ Index + Middle up | Lift pen (move without drawing) |
| 🤏 Pinch (thumb + index) | Erase |
| 🖐️ Open hand (hold ~0.6s) | Clear canvas |

## 🖌 Brush Features

- **4 styles**: Round · Square · Spray · Neon Glow ⚡
- **10 preset colors** + custom color picker
- Adjustable **size** (2–60px) and **opacity**
- **Undo** (30 steps), **Save PNG**, **Mirror** toggle

## 🛠 Tech Stack

- **MediaPipe Hands** (CDN) — real-time 21-landmark hand tracking
- **HTML5 Canvas 2D** — drawing engine
- **Vanilla JS** — no framework, no build tool needed
